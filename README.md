# Detalles del examen
Este examen tiene 4 niveles, con una pregunta elegida al azar de cada nivel.

Si no aprueba el examen, cuando vuelva a realizarlo, comenzará de nuevo en el Nivel 1.

No hay Norminette en este examen.
